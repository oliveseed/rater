# Rater
A platform to rate or leave reviews on things. Features a login system, recommendation system, and search. Backend is PHP/MySQL and frontend is vanilla HTML/JS with Tailwind CSS.

## Example deployment
### Home page
Features:
- see trending items
- see latest activity
- search bar
